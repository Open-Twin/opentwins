Setup

Running in under 5 minutes

Install, walk through a web wizard, and your first agent starts engaging.

1

Install & configure

npm install -g opentwins
opentwins init

A web wizard guides you through identity, platforms, voice, and schedule. No config files to edit.

2

Connect platforms

In the dashboard:
Agents → Set up browser
Log in → Confirm

One click launches Chrome. Log in to each platform once — sessions persist across runs.

3

Watch it grow

opentwins start --ui

Capabilities

Everything under the hood

🤖 Claude-powered agents

Each platform gets a dedicated AI agent that reads context, thinks about what to say, and engages authentically.

📊 7-stage content pipeline

Daily trend scouting, competitive intel, engagement tracking, amplification, and content generation.

🎯 Voice calibration

Style mix, disagreement targets, word limits, and brand mention frequency keep your twins sounding like you.

📈 Quality monitoring

Track style distribution, disagreement rates, word density over 7/14/30 day windows.

💰 Cost tracking

Per-agent tokens, cache hit rates, estimated spend, daily trends. Know exactly where every dollar goes.

🛡️ Safety first

Rate limits, active hours, banned phrases. Runs locally on your machine — no third-party access.
